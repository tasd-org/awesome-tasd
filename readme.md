# Awesome TASD

A curated list of resources and tools for creating, modifying, and using [TASD](https://tasd.io/) files. TASD is a file format used for storing Tool Assisted Speedrun input and configuration data, as well as related metadata, for use with TAS replay devices.

If you want to contribute, please read [this](contributing.md).

Go to [tasd.io](https://tasd.io/) for the latest TASD specification. The source can be found [here](https://github.com/tasd-org/tasd-spec).

## Documentation
